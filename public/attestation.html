Application de création d'attestations de précarité

Description

Certains étudiants sont dans des situations de précarité mais n'ont pas accès à certaines aides. Cependant le Crous met à leur disposition des aménagements en leur conférant un statut prouvé par une attestation. Ma mission était donc de créé une application permettant facilement de créer des attestations à partir d'un fichier csv, de voir les détails de celles dans la base et pouvoir les supprimer. L'application permet aussi de créer un nouveau fichier csv regroupant nom et prenom de chaque étudiant bénéficiant d'une attestation. Ce fichier contient aussi deux lien : l'un renvoie directement sur une page affichant l'attestation, le second vers un Qrcode qui ramene à l'attestation. Le Qrcode est créé à l'import du fichier csv qui construit les attestations.

Compétences mobilisées

  • Base de Données

Travail personnel
