Application de demande de collocations

Description

Le Crous met à la disposition des étudiants des logements pour la collocations. Dans le but de facilité les demandes et l'attribution de ces collocations, ma mission était de créer une application qui puisse permettre aux étudiants de faire une demande de collocation en entrant ses informations dans la base. Les agents du Crous eux ont accés à une interface permettant de gérer la base de données en fonction de leurs permissions.

Compétences mobilisées

  • Symfony
  • Base de Données
  • Gestion de Projet

Travail personnel

Création en entier de l'application, création et gestion de la base de données, création d'une interface admin avec easyadmin
